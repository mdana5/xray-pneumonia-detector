# Medical Imaging Quality Assurance System – FastAPI App


This directory contains the FastAPI backend for the Medical Imaging Quality Assurance System.  
It enables users to upload chest X-ray images, get AI-based predictions, and view heatmaps for diagnosis explanation.

---

##  Features

- Upload chest X-ray images (JPG, PNG)
- Deep Learning prediction (Normal / Pneumonia)
- Heatmap generation for explainability
- REST API using FastAPI
- Simple frontend using streamlit 
- Docker support for deployment
